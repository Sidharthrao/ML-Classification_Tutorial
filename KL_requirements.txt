# ============================================================================
# TROUBLESHOOTING COMMON ISSUES
# ============================================================================
#
# Issue: "No module named 'sklearn'"
# Solution: Install scikit-learn (not sklearn package)
#
# Issue: LightGBM installation fails on Windows
# Solution: Install Visual C++ Build Tools or use conda
#
# Issue: SHAP installation issues
# Solution: Ensure scikit-learn is installed first
#
# Issue: Jupyter kernel not found
# Solution: python -m ipykernel install --user
#
# Issue: Import errors after installation
# Solution: Restart Python kernel/IDE
#
# ============================================================================
# MEMORY & PERFORMANCE CONSIDERATIONS
# ============================================================================
#
# For Large Datasets (6M+ rows):
# - Use chunked processing in pandas (chunksize parameter)
# - Consider dask for out-of-core computation (optional)
# - Use pyarrow/parquet for efficient storage
# - Monitor memory with memory_profiler (optional)
#
# For Faster Training:
# - Ensure numpy/scipy use optimized BLAS (MKL, OpenBLAS)
# - Use n_jobs=-1 for parallel processing
# - Consider GPU acceleration for deep learning
